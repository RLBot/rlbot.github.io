RLBot

Write custom Rocket League bots for offline play.

What is RLBot?

RLBot is a framework that allows programmers to create offline Rocket League bots. We supports a variety of languages and fits all programming skill levels. As a community we frequently hold bot tournaments where bot makers compete by making bots and pit them against each other.
